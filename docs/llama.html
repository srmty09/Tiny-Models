Tiny Llama v1

72M params

A Llama-style causal language model implemented from scratch in PyTorch. Decoder-only transformer with Grouped Query Attention, Rotary Position Embeddings, RMSNorm, SwiGLU activation, and KV-cache streaming generation with tied embeddings.

Weights Source Code W&B Run

Key Features

Grouped Query Attention
12 query heads with 4 KV heads for memory-efficient attention
Rotary Embeddings
RoPE with theta 10K for relative position encoding
SwiGLU MLP
Gated feed-forward with SiLU activation and 1024 intermediate dim
RMSNorm
Pre-norm architecture for stable gradient flow
KV Cache
Prefill + decode for efficient autoregressive generation
Weight Tying
Shared embedding and output projection weights
